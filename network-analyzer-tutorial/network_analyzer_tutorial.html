①VNA（ネットワークアナライザー）とは

VNA（Vector Network Analyzer）は、RF回路に信号を送り込み、 どのくらい反射するか・どのくらい透過するかを 周波数ごとに測定する装置です。
測定結果は Sパラメータ（散乱行列）として表され、 フィルタ・アンプ・コネクタ・基板配線など、あらゆるRF部品の特性評価に使われます。

③Sパラメータとは

Sパラメータ（散乱パラメータ）は、RFポートに入力した波が 「どれだけ反射し、どれだけ透過するか」を周波数ごとに表した複素数です。
ここでは最もよく使う 2ポート回路（入力1ポート・出力1ポート）に絞って説明します。

2ポートDUT — Port1 から信号を入れた場合
a と b — 電力波の定義
a（入射波）：ポートへ入ってくる波。単位 √W
b（出射波）：ポートから出ていく波。単位 √W
|a|² = 入射電力 [W]、|b|² = 反射/透過電力 [W]
例：−10 dBm 入力 → |a| ≈ √100 μW ≈ 10 mV/√Ω
S パラメータの定義
S₁₁ = b₁ / a₁ （a₂=0 のとき）
S₂₁ = b₂ / a₁ （a₂=0 のとき）
Sij は無次元の複素数（振幅＋位相）。
受動素子では |Sij| ≤ 1（エネルギー保存より）。
通常は 20 log|Sij| [dB] で表示。
パラメータ意味0 dB / −∞ dB
S₁₁ 反射係数
リターンロス		 全反射（短絡/開放）/ 完全整合（反射ゼロ）
S₂₁ 透過係数
挿入損失 / 利得		 損失なし透過 / 完全遮断（信号ゼロ）

④キャリブレーション（校正）

VNAとDUTの間には必ずケーブルやコネクタが介在します。 このためケーブルの反射・損失・結合器の指向性誤差・インピーダンス不整合 など「系統誤差」が測定値に混入します。
キャリブレーション（校正）は、特性が既知の標準器（Cal Kit）を使って これらの誤差係数を実測し、以降の測定からリアルタイムに差し引く操作です。
測定前に必ず実施すること。キャリブレーションなしの測定値は信頼できません。

1STEP
SHORT（短絡）を Port1 / Port2 に接続
Port1・Port2 それぞれの先端にショート標準器（金属キャップで短絡）を取り付けます。信号は全部反射して戻り、位相が180°ひっくり返ります（Γ = −1）。
→ VNAは「理想の全反射」と実際の測定値のずれを記録。方向性誤差（ED: Directivity）と反射トラッキング誤差（ER: Reflection Tracking）の算出に使われます。
2STEP
OPEN（開放）を Port1 / Port2 に接続
Port1・Port2 それぞれの先端を何もつながない（開放）にします。こちらも全反射ですが位相は変わりません（Γ = +1）。
→ SHORTと2点セットで測ることで、反射トラッキング誤差（ER: Reflection Tracking）が確定。さらにソース整合誤差（ES: Source Match）の算出も開始されます。
3STEP
LOAD（整合終端）を Port1 / Port2 に接続
Port1・Port2 それぞれに精密な 50 Ω ターミネーターを接続します。信号がすべて吸収されて反射ゼロが理想（Γ = 0）。
→ 残留する微小な反射から方向性誤差（ED）が直接確定し、ソース整合誤差（ES）も確定。SHORT / OPEN / LOAD の3点で Port1・Port2 それぞれの ED / ES / ER（計6項）がすべて揃います。
4STEP
THROUGH（スルー）を接続
Port1とPort2を直結します（理想では S21 = 0 dB：ケーブルがなければ損失ゼロ）。
→ 実測とのずれから順方向・逆方向それぞれの透過トラッキング誤差（ET: Transmission Tracking）・負荷整合誤差（EL: Load Match）・漏れ誤差（EX: Isolation）（計6項）が確定。これで ED / ES / ER / ET / EL / EX（各2方向）= 合計12項すべて確定し、2ポートキャリブレーション完了です。
✓
4ステップ完了 → VNAが 12誤差係数
なぜ「12個」？ — 2ポート誤差モデル
① Port1 側の反射誤差（3項）
ED1：方向性誤差結合器の「漏れ」成分
ES1：ソース整合誤差VNA出力のインピーダンスずれ
ER1：反射トラッキング利得・位相の周波数依存ずれ
② Port2 側の反射誤差（3項）
ED2・ES2・ER2：Port2 に対する同様の3誤差
③ 順方向（Port1→Port2）透過誤差（3項）
ET21：透過トラッキングPort1→Port2 の利得ずれ
EL2：負荷整合誤差Port2 入力のインピーダンスずれ
EX21：漏れ成分ケーブルを介した直接漏れ
④ 逆方向（Port2→Port1）透過誤差（3項）
ET12・EL1・EX12：逆方向の同様の3誤差
SHORT / OPEN / LOAD を Port1 と Port2 それぞれに適用 → 計6項確定
THROUGH（直結）で残り6項を確定 → 合計12項

⑤まとめ
VNAの役割
RF部品に信号を入力し、反射波・透過波をSパラメータとして周波数掃引測定する装置。
S11 / S21
S11＝反射特性。S21＝透過特性。値が0 dBに近いほど全反射（S11）または無損失透過（S21）。
方向性結合器
進行波と反射波を方向別に分離する部品。有限の方向性（Directivity）が測定精度の限界になる。
キャリブレーション
SHORT / OPEN / LOAD / THROUGH の既知標準で系統誤差を除去。測定前に必ず実施すること。
インピーダンス整合
Z_L = Z₀ = 50 Ω のとき Γ = 0 で反射なし。RF設計の基本はインピーダンスを揃えること。
反射係数 Γ
Γ = (Z_L − Z₀) / (Z_L + Z₀)。S11 [dB] = 20 log₁₀|Γ|。整合時 −∞ dB、全反射時 0 dB。
⑥実践測定例 — 代表的 DUT の S パラメータ

実際の VNA 測定でよく登場する DUT の S パラメータ特性を俯瞰します。 波形の「形」を覚えると、測定結果を見た瞬間に「整合が取れていない」「ここにフィルタが入っている」と 直感的に判断できるようになります。 青線 = S11（反射）、 橙線 = S21（透過・利得）、 緑線 = S21（利得 > 0 dB）。
