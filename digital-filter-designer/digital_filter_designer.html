デジタルフィルター設計ツール

LPF / HPF / BPF — 周波数特性・インパルス応答・リアルタイム可視化

① デジタルフィルターとは？

デジタルフィルターは、信号の特定の周波数成分を通す・遮断する処理です。
ノイズ除去・信号抽出・DC オフセット除去など、DSP の基本演算として広く使われます。
DSP / マイコン / FPGA のデジタル回路で実装され、アナログフィルターより設計が容易です。

オーディオノイズ除去
録音信号から高周波ノイズを除去。LPF で 20 kHz 以上をカット。可聴域をクリーンに保つ。
センサーデータ平滑化
加速度センサー・温度センサーの高周波ノイズを LPF で除去。安定した計測値を得る。
通信信号処理
BPF で目的チャンネルの周波数帯域だけを抽出。不要な隣接チャンネルを遮断する。
DC オフセット除去
HPF でゼロ Hz 付近（DC 成分）を遮断。センサーのオフセット電圧を取り除く。

② ローパスフィルター（LPF）とは？

低い周波数は通す、高い周波数は遮断します。「ノイズ除去」「平滑化」に最もよく使われるフィルターです。
カットオフ周波数（fc）が境界線で、それより低い成分はそのまま通過し、高い成分は周波数が上がるにつれて徐々に弱まっていきます。 例えるなら「fc より高い音は、周波数が倍になるたびに音量を半分ずつ下げる」イメージです。
次数（N）を上げると遮断が急になります。1次だと緩やかにしか遮断できませんが、4次・8次になると fc 付近でほぼ壁のようにバッサリ遮断できます。

③ ハイパスフィルター（HPF）とは？

高い周波数は通す、低い周波数は遮断します。信号に混ざり込んだ「ゆっくりした変動」を取り除くのに使われます。
よくある例は DC オフセット除去です。マイクやセンサーの出力信号に「ずっと +0.5V ずれた状態」が乗ることがあります。これはゼロ Hz 成分（直流）なので、HPF を通すだけでスパッと除去できます。
他にも「電源から来る 50/60 Hz ハム音」「体重計の揺れのような超低周波ノイズ」など、変化がゆっくりな成分を除去したい場面で活躍します。

HPF — ビフォー / アフター波形
入力（DC オフセット + 高周波）
HPF 出力（DC 除去）

④ バンドパスフィルター（BPF）とは？

特定の周波数範囲だけを通すフィルターです。通信・計測で特定周波数の信号抽出に使われます。
中心周波数 f0 と 帯域幅 BW で定義されます。LPF と HPF を組み合わせた構造です。
Q 値（= f0 / BW）が高いほど選択性が上がります。

BPF — ビフォー / アフター波形
入力（低・中・高周波混在）
BPF 出力（中心周波数のみ）

⑤ 振幅の減衰特性と位相の遅れ

左 Y 軸（実線）＝振幅 [dB]：0 dB がそのまま通過、マイナスになるほど信号が弱くなります。
右 Y 軸（破線）＝位相 [deg]：フィルターを通ると信号が「遅れて」出てくる量を角度で示します。
実際の計測器（ネットワークアナライザーなど）でも、この 2 軸を同一グラフに重ねて表示するのが標準的なボード線図のスタイルです。
位相は ±180° で折り返し（ラップアラウンド）します。BPF の位相が +180° → 0° → −180° と推移するのが見てとれます。

⑦ フィルター次数の影響：遮断特性

次数が高いほど遮断特性が急峻になります。1次 = −20 dB/decade、2次 = −40 dB/decade と増えます。
高次ほど計算量が多く、遅延（群遅延）も増えます。リアルタイム処理では次数の選択が重要です。
Butterworth フィルターは通過域が最大限平坦で、実用的によく使われます。

⑧ インパルス応答：時間領域での動作

インパルス応答はフィルターへのパルス入力に対する出力で、時間領域の特性を示します。
LPF は滑らかな応答、HPF は尖った形、BPF は振動的な応答（共振周波数が見える）になります。
現在のスライダー設定（タイプ・fc・次数）に連動してリアルタイムで更新されます。

⑨ FIR フィルターの実装（Direct Form I）

FIR（有限インパルス応答）フィルターは、シフトレジスタと係数の積和演算（MAC）で実現されるシンプルなデジタルフィルターです。
各ステップで新しいサンプルが入力され、レジスタ値が右にシフト、各係数との積を総和して出力 y[n] を得ます。
FPGA / DSP / マイコンに直接マッピングでき、線形位相特性が得られる点が特徴です。

⑩ 実例：オーディオフィルター

オーディオでよく使われる LPF：高周波ノイズ（ハム音・量子化ノイズ）の除去に使います。
人間の可聴域は約 20 Hz ～ 20 kHz。カットオフ 20 kHz の LPF で可聴域外のノイズを除去します。
fc = 10 kHz の LPF（2次）なら、10 kHz 以上の成分を −40 dB/decade で遮断します。
