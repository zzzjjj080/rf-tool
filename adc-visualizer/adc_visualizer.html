ADCとは何か？

ADC（アナログ-デジタル変換器）は、連続したアナログ信号を サンプリング（時間離散化）と量子化（振幅離散化）の 2ステップでデジタルデータに変換します。 サンプリングレート・ビット数・入力モード・コード方式の設定が出力品質を決定します。

理論的背景

ナイキスト定理
エリアシングを防ぐには fs ≥ 2·fin が必要。 fs < 2·fin の場合、入力信号は |fin mod fs| の周波数として折り返されます。 f_alias = |fin − round(fin/fs)·fs|
量子化ノイズ
量子化誤差は ±LSB/2 の範囲に一様分布。RMS ノイズは LSB/√12。 SQNR = 6.02·N + 1.76 dB 入力がフルスケールより小さいと実効 SNR は低下します。
ENOB
実効ビット数（ENOB）は理想 ADC に換算したビット数。 Ain < Vref の場合、使用ビット数が減少し実効解像度が下がります。 ENOB = (SNR − 1.76) / 6.02
差動入力モード
Fully-Diff では IN+ と IN- が対称に振れ、実効フルスケールが 2×Vref（差動）になります。 Pseudo-Diff は IN- を固定（Vcm）にした簡易版。 Fully-Diff: LSB = 4·Vref / 2ᴺ
コード方式
2の補数: 入力 Vcm → コード 0x0。ゼロ基準バイポーラ表現。
Offset Binary: 入力 Vcm−Vref → コード 0x0、Vcm → 中間コード。 AD9xxx は原則 2の補数出力
クリッピング / Vcm
Ain > Vref の場合、信号が ±Vref でクリップされ激しい歪みが発生します。 Vcm（コモンモード電圧）は入力レンジの中心電圧で、ADC によって 0V〜AVDD/2 など異なります。 理想: Ain ≈ 0.7〜1.0 × Vref
