Phase Locked Loop — ロック動作をインタラクティブに学ぶ
① PLL とは？— 応用例から理解する

PLL（フェーズロックドループ）は、入力信号の周波数・位相に同期した出力を生成する回路です。

低速な基準クロックを高速クロックに変換したり、受信信号に位相を合わせる用途など、デジタル・RF 回路の至る所で使われています。

下の3つが代表的な応用例です。

🔲
FPGA / ASIC
外部 25 MHz 水晶から内部 500 MHz クロックを生成。電源投入後 PLL がロックするまでリセット状態を保持。
📡
通信 IC / RF
受信キャリアに位相同期し、雑音の多い環境でも正確に復調。GPS・LTE・Wi-Fi で必須の技術。
⏱️
クロックジェネレーター
1つの基準クロックから 100 MHz / 200 MHz / 50 MHz など複数の周波数を同時生成。ボード全体の同期基準に。
② PLL の目的 — なぜ必要か

デジタル回路はすべてのフリップフロップが同一クロックに同期して動く必要があります。ズレが生じると「セットアップ違反」や「メタスタビリティ」が発生します。

PLL は基準信号に対して位相を一致（ロック）させることで、システム全体の時間基準を正確に揃えます。

また入力より高い周波数を出力できるため、低速な水晶から高速クロックを合成する用途にも不可欠です。

✗ PLL なし
各ブロックのクロックがバラバラ
→ タイミング違反・データ化け
→ 誤動作・システムクラッシュ
高速クロックが作れない
✓ PLL あり
全ブロックが同一位相のクロックで動作
→ データが正確に受け渡し
→ 安定した高速動作を実現
基準から任意倍率の周波数を生成
③ PLL の構成 — ブロック図

PLL は 4つの要素で構成されます：位相比較器（PD） → ループフィルター（LF） → VCO → 分周器（÷N）

VCO（電圧制御発振器）：制御電圧が上がると発振周波数が上がる素子。PLL の心臓部です。

出力を分周器で 1/N に分周してフィードバックし、基準信号と位相比較することで周波数を N 倍に保ちます。

── 調整メカニズム（1ループの因果連鎖）──
① PD：Fref と VCO÷N の位相差を電圧として出力（差が大きいほど大きな電圧）
② LF：PD 出力の高周波ノイズを除去し、滑らかな直流制御電圧に変換
③ VCO：制御電圧に比例して発振周波数を上下（高電圧→高周波）
④ ÷N：VCO 出力を N で割って PD へ帰還 → ループ完結
位相差 → 電圧 → 周波数変化 → 位相差縮小 … を繰り返してロックへ収束
④ 動作原理 — 位相差を検出して周波数を調整

4 本の信号を同時に観察することで、PLL がどうやって位相を合わせるかが見えます。① 基準信号・② 帰還信号（VCO÷N）の位相差を③ 位相比較器（PD）が検出し、④ ループフィルター（LF）が平滑化した制御電圧を VCO に送り込みます。

PD 出力の振幅は位相差に比例 — じわじわと振れ幅が小さくなるのがロックしていく証拠です。

制御電圧が定常値に落ち着いたとき、VCO の周波数が目標値に固定され「ロック完了」となります。

⑥ ループゲインの影響 — 応答速度 vs 安定性

ループゲイン大：ロックが速い（応答性良い）が、行き過ぎてオーバーシュート・発振の危険があります。

ループゲイン小：ロックが遅い（応答性悪い）が、安定して収束します。

設計では用途に応じてダンピング比 ζ ≈ 0.7（臨界減衰に近い）を狙うのが一般的です。

ゲイン 大（K=8, 過減衰なし）
ゲイン 中（K=3, 推奨）
ゲイン 小（K=1, 安定・低速）
⑦ 周波数倍率 — 分周比の役割

「分周（遅くする）しているのに、なぜ基準より速い周波数が出るの？」— PLL で最も不思議に感じる点です。

答えはフィードバックにあります。PLL がロックしているとき位相比較器の両入力は必ず同じ周波数でなければなりません。帰還側は VCO を N で割った信号なので、等式が成り立つよう VCO が自動調整されます。

つまり「分周して基準に合わせる」ことで、結果として VCO = N × 基準 という関係が強制されます。

── ループの等式（なぜ N 倍になるか）──
ロック条件：Fref = Fvco ÷ N　（PD 両入力が同周波数）
∴ Fvco = N × Fref

もし Fvco が低下 → Fvco/N < Fref → PD が「遅れ」検出
→ 制御電圧 ↑ → VCO 周波数 ↑ → 元の N×Fref に戻る

直接周波数を作っているのではなく、フィードバックが VCO を強制している

⑧ 実例 — クロックジェネレーター

ボード設計では 1 つの水晶発振器（例: 25 MHz）から複数の PLL を使い異なる周波数を同時供給します。

FPGA コア用 200 MHz、DDR メモリ用 400 MHz、USB 用 480 MHz — それぞれ別の分周比を持つ PLL が担当します。

全 PLL が共通の基準に同期しているため、クロックドメイン間のデータ転送も安全に行えます。
