📖 ABCD 各要素の意味
A
出力端短絡時の電圧比（V₁/V₂）。A=1 なら電圧変換なし（直列素子のみ）。単位なし。
B
出力端短絡時の転送インピーダンス（V₁/I₂）。B=0 なら直列成分ゼロ（並列素子のみ）。単位：Ω。
C
出力端開放時の転送アドミタンス（I₁/V₂）。C=0 なら並列成分ゼロ（直列素子のみ）。単位：S。
D
出力端開放時の電流比（I₁/I₂）。D=1 なら電流変換なし（並列素子のみ）。単位なし。
⛓ カスケード接続 — 行列の積

Mtotal = M1 × M2 × M3
これがSパラメータにはない「カスケードの強み」です。

📝 変換式
ABCD → S（Z₀ 基準） denom = A + B/Z₀ + C·Z₀ + D
S₁₁ = (A + B/Z₀ − C·Z₀ − D) / denom
S₂₁ = 2 / denom
S₁₂ = 2·(AD − BC) / denom
S₂₂ = (−A + B/Z₀ − C·Z₀ + D) / denom
S → ABCD（Z₀ 基準） A = [(1+S₁₁)(1−S₂₂) + S₁₂S₂₁] / 2S₂₁
B = Z₀·[(1+S₁₁)(1+S₂₂) − S₁₂S₂₁] / 2S₂₁
C = [(1−S₁₁)(1−S₂₂) − S₁₂S₂₁] / (2S₂₁·Z₀)
D = [(1−S₁₁)(1+S₂₂) + S₁₂S₂₁] / 2S₂₁

De-embeddingとは？
実測値には「治具（ケーブル・コネクタ・基板パターン）」の影響が含まれています。 治具のABCD行列で逆算することで、DUT（被測定物）の真の特性を取り出す技術です。
測定値 M = FA × DUT × FB  →  DUT = FA⁻¹ × M × FB⁻¹
これはVIセンサー補正の原理でもあります。ケーブルや配線のインダクタンス・容量を除去することで、 プラズマ負荷の真のインピーダンスが得られます。
