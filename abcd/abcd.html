📡 Sパラメータ（散乱行列）とは？

S パラメータ（Scattering Parameters）は、RF 回路・マイクロ波回路で最もよく使われる特性表現です。 「信号がどれだけ反射されるか（S₁₁）」「どれだけ通り抜けるか（S₂₁）」を 複素数（大きさ＋位相）で表します。

ネットワークアナライザー（VNA）が直接測定するのは S パラメータです。 インピーダンス Z やアドミタンス Y と違い、S パラメータは 基準インピーダンス Z₀（通常 50 Ω）に対する入射波・反射波・透過波の比として定義されます。

まず具体的な数値で S パラメータの「感覚」をつかんでから、ABCD 行列の話に進みましょう。

🔁 S₁₁（反射係数 Γ）
S₁₁ の定義
  • ポート1から入れた波のうち、跳ね返ってくる割合
  • |S₁₁| = 0 → 完全整合（反射ゼロ、最高の状態）
  • |S₁₁| = 1 → 全反射（開放・短絡など）
  • スミスチャートにプロットされるのはこの S₁₁（= Γ）
1 ポート負荷の場合 S₁₁ = Γ = (Z_L − Z₀) / (Z_L + Z₀)
➡ S₂₁（透過係数）
S₂₁ の定義
  • ポート1から入れた波のうち、ポート2へ抜けていく割合
  • |S₂₁| = 1 → 完全透過（0 dB）
  • |S₂₁| = 0.707 → −3 dB（電力が半分になる点）
  • |S₂₁| ≈ 0 → 全反射または全損失
受動回路の電力保存則 |S₁₁|² + |S₂₁|² ≤ 1（等号：無損失回路）

スミスチャート上の点をドラッグするか、スライダーで負荷 Z_L を変えてみてください。 Z_L = Z₀ = 50 Ω のとき S₁₁ = 0（完全整合・チャート中心）になります。

🚀 次のステップへ：なぜ ABCD 行列が必要か？
S パラメータの限界：複数の 2ポート回路を直列につなぐとき、 S パラメータで合成しようとすると非常に複雑な式（シグナルフロウグラフ）が必要になります。
ABCD 行列の利点：直列（カスケード）接続のとき、 ABCD 行列は単純な 行列の積 M_total = M₁ × M₂ × M₃ × … で合成できます。
もちろん S パラメータと ABCD 行列は完全に相互変換できるので、 VNA で測定した S パラメータも ABCD に変換して使えます。

S パラメータと違い、ABCD 行列は 回路を直列につないだときに行列の積で合成できる という大きな利点があります。 ケーブル・フィルタ・増幅器などを組み合わせた系全体の特性を、ブロックごとに計算して掛け合わせるだけで求められます。

⛓ カスケード接続とは？ — なぜ行列の積で合成できるのか

複数の 2ポート回路ブロックを 直列（カスケード）につないだとき、合成 ABCD 行列は各ブロックの行列積 で求まります。

M_total = M₁ × M₂ × M₃ × …

これは ABCD 行列の最大の利点です。S パラメータでは複数のブロックを合成するのに複雑な式が必要ですが、 ABCD 行列なら 単純な行列の掛け算だけ で済みます。 たとえばケーブル → フィルタ → 増幅器の合成特性も、それぞれの ABCD 行列を掛け合わせるだけです。

🔄 S パラメータと ABCD 行列の相互変換

ネットワークアナライザー（VNA）が測定するのは S パラメータ（S11, S21 など）です。 一方、回路解析では ABCD 行列 が便利です。この 2 つは完全に相互変換できます。

変換には 基準インピーダンス Z₀（通常 50 Ω）が必要です。 S パラメータは「Z₀に対する反射・透過」を表すため、Z₀ が変われば同じ回路でも S パラメータの値が変わります。 一方 ABCD 行列は Z₀ に依存しません。

下で回路タイプと素子値を変えながら、S パラメータと ABCD 行列が同時に更新されるのを確認してください。

📝 変換式（参考）
ABCD → S（Z₀ 基準） denom = A + B/Z₀ + C·Z₀ + D
S₁₁ = (A + B/Z₀ − C·Z₀ − D) / denom
S₂₁ = 2 / denom
S₁₂ = 2·(AD − BC) / denom
S₂₂ = (−A + B/Z₀ − C·Z₀ + D) / denom
S → ABCD（Z₀ 基準） A = [(1+S₁₁)(1−S₂₂) + S₁₂S₂₁] / 2S₂₁
B = Z₀·[(1+S₁₁)(1+S₂₂) − S₁₂S₂₁] / 2S₂₁
C = [(1−S₁₁)(1−S₂₂) − S₁₂S₂₁] / (2S₂₁·Z₀)
D = [(1−S₁₁)(1+S₂₂) + S₁₂S₂₁] / 2S₂₁
注意：S → ABCD 変換では S₂₁ ≠ 0 が必要です（開放/短絡回路では使えません）。
🔢 ABCD パラメータ（Z₀ 不依存）
行列要素の読み方
  • A, D ≈ 1 → 主に「透過」な回路
  • B が大きい → 大きな直列インピーダンス（電圧降下）
  • C が大きい → 大きな並列アドミタンス（電流の漏れ）
📡 S パラメータ（Z₀ 依存）
S パラメータの意味
  • |S₁₁|：ポート1 での反射係数（0 = 無反射、1 = 全反射）
  • |S₂₁|：ポート1→2 への透過係数（1 = 完全通過、0 = 全損失）
  • |S₁₁|² + |S₂₁|² ≤ 1（受動回路では電力保存則が成立）
🔬 De-embedding（脱埋め込み）とは？

実際の測定では、被測定物（DUT: Device Under Test）の前後に ケーブル・コネクタ・基板パターンなどの 治具（fixture）が接続されています。 VNA が測定するのは「DUT + 治具」込みの特性です。

VNA ── [治具A（FA）] ── [DUT] ── [治具B（FB）] ── VNA
測定値 M = FA × DUT × FB

治具の ABCD 行列が分かっていれば、逆行列演算で DUT の真の特性を取り出せます：

DUT = FA⁻¹ × M × FB⁻¹

これは VIセンサーによるインピーダンス補正 の原理でもあります。 ケーブルや CT・電圧プローブの特性（インダクタンス・容量）を de-embedding することで、 プラズマ負荷の真のインピーダンスを計算できます。

治具インダクタンスの影響
治具の直列 L が増えると、スミスチャート上の点が 時計回りに回転（位相遅れ）します。 de-embedding はこの回転を逆方向に戻す操作に相当します。
