Cascade Matrix · S↔ABCD Conversion · De-embedding Principle
📡 Sパラメータ（散乱行列）とは？
  • RF・マイクロ波回路で最もよく使われる特性表現 — VNA（ネットワークアナライザー）が直接測定する量
  • S₁₁（反射係数）：ポート1から入れた信号が跳ね返ってくる割合
  • S₂₁（透過係数）：ポート1から入れた信号がポート2へ抜ける割合
  • 大きさと位相をセットにした複素数で表す（例：0.686∠47°）
  • 基準インピーダンス Z₀（通常 50 Ω）に対する入射波・反射波・透過波の比として定義

まずは具体的な数値で S パラメータの感覚をつかんでから、ABCD 行列へ進みましょう。

🔁 S₁₁（反射係数 Γ）
S₁₁ の定義
  • ポート1から入れた波のうち、跳ね返ってくる割合
  • |S₁₁| = 0 → 完全整合（反射ゼロ、最高の状態）
  • |S₁₁| = 1 → 全反射（開放・短絡など）
  • スミスチャートにプロットされるのはこの S₁₁（= Γ）
1 ポート負荷の場合 S₁₁ = Γ = (Z_L − Z₀) / (Z_L + Z₀)
➡ S₂₁（透過係数）
S₂₁ の定義
  • ポート1から入れた波のうち、ポート2へ抜けていく割合
  • |S₂₁| = 1 → 完全透過（0 dB）
  • |S₂₁| = 0.707 → −3 dB（電力が半分になる点）
  • |S₂₁| ≈ 0 → 全反射または全損失
受動回路の電力保存則 |S₁₁|² + |S₂₁|² ≤ 1（等号：無損失回路）

🚀 次のステップへ：なぜ ABCD 行列が必要か？
S パラメータの限界
  • 複数の 2ポート回路を直列につなぐと、S パラメータでの合成には複雑な式（シグナルフロウグラフ）が必要
  • ブロックが増えるほど計算が爆発的に複雑になる
ABCD 行列の利点
  • 直列接続 → 行列の積 M_total = M₁ × M₂ × M₃ × … で一発合成
  • S パラメータと完全に相互変換可能 → VNA の測定値をそのまま活用できる
  • Z₀ に依存しないため、異なる測定系のデータも統一的に扱える
ABCD行列（カスケード行列）とは？

2ポート回路（入力端子と出力端子を持つ回路）の特性を表す方法の一つが ABCD行列（または伝達行列）です。 入力側の電圧・電流と出力側の電圧・電流を次の式で結びつけます：

[
V₁ I₁
]
=
[
A B C D
]
×
[
V₂ −I₂
]

S パラメータと違い、ABCD 行列は 回路を直列につないだときに行列の積で合成できる という大きな利点があります。 ケーブル・フィルタ・増幅器などを組み合わせた系全体の特性を、ブロックごとに計算して掛け合わせるだけで求められます。

⛓ 直列接続とは？ — なぜ行列の積で合成できるのか
  • 複数の 2ポート回路を直列につなぐ → 合成 ABCD 行列は各ブロックの行列積で求まる
  • M_total = M₁ × M₂ × M₃ × …
  • S パラメータでの合成は複雑（シグナルフロウグラフが必要）だが、ABCD なら行列の掛け算だけ
  • ケーブル → フィルタ → 増幅器のような多段構成も、それぞれの ABCD を掛け合わせるだけ
💡 M という記号について： ABCD 行列は「伝達行列（Transmission Matrix）」とも呼ばれます。 このページでは各ブロックの ABCD 行列を M₁, M₂, M₃（Matrix の頭文字＋番号）と表記しています。 M_total はそれらを掛け合わせた合成行列です。

3 つのブロックのタイプ・値・周波数を変えて、合成結果がどう変わるか確認してください。

🔲 ABCD 行列とは？ — S パラメータとの相互変換
  • ABCD 行列（伝達行列）— 2ポート回路の入出力ポートの電圧・電流を結びつける 2×2 行列
  • Z₀（基準インピーダンス）に依存しない → 測定系が違っても同じ値になる
  • S パラメータと完全に相互変換可能（変換には Z₀ が必要）

V₁, I₁ = ポート1（入力側）の電圧・電流 V₂, I₂ = ポート2（出力側）の電圧・電流 −I₂：出力電流の符号規約（流れ出る向きを正とする）

📖 A・B・C・D 各要素の意味
📌 ポート変数： V₁・I₁ = 入力ポート（ポート1）の電圧・電流 ／ V₂・I₂ = 出力ポート（ポート2）の電圧・電流 （I はポートへ流れ込む向きを正とする）
A
電圧伝達比 ／ 無次元
ポート2 開放時（I₂=0）の電圧比 V₁/V₂
出力側を「何も接続しない」状態にしたとき、入力電圧 V₁ と出力電圧 V₂ の比。 信号が電圧としてどれだけ通り抜けるかを示す。
A=1 → 電圧がそのまま伝わる（直列素子だけの回路はつねに A=1）
A≠1 になるのはトランスなど電圧変換が起きるときだけ
直列Z：A = 1　／　並列Y：A = 1
B
転送インピーダンス ／ Ω
ポート2 短絡時（V₂=0）の転送インピーダンス V₁/I₂
出力側を「ショート」したとき、出口に流れる電流 I₂ と入力電圧 V₁ の比（単位 Ω）。 直列に入っているインピーダンスの大きさがそのまま現れる。
B=Z → 直列素子のインピーダンス値（L なら B=jωL）
B=0 → 直列成分ゼロ（並列素子だけの回路）
直列Z：B = Z　／　並列Y：B = 0
C
転送アドミタンス ／ S（＝1/Ω）
ポート2 開放時（I₂=0）の転送アドミタンス I₁/V₂
出力側を「開放」したとき、出口電圧 V₂ と入力電流 I₁ の比（単位 S）。 横に分流するアドミタンスの大きさがそのまま現れる。
C=Y → 並列素子のアドミタンス値（C なら C=jωC）
C=0 → 並列成分ゼロ（直列素子だけの回路）
直列Z：C = 0　／　並列Y：C = Y
D
電流伝達比 ／ 無次元
ポート2 短絡時（V₂=0）の電流比 I₁/I₂
出力側を「ショート」したとき、入力電流 I₁ と出力電流 I₂ の比。 電流がどれだけそのまま通り抜けるかを示す。
D=1 → 電流がそのまま通過（並列素子だけの回路はつねに D=1）
D≠1 になるのはトランスなど電流変換が起きるときだけ
直列Z：D = 1　／　並列Y：D = 1
💡 パターン早見： 直列インピーダンスZのみ → A=D=1, B=Z, C=0　／ 並列アドミタンスYのみ → A=D=1, B=0, C=Y　／ 行列式 AD−BC=1（受動可逆回路の条件）
▶ おまけ：ABCD → S 逆変換式（Z₀ が必要）
ABCD → S（Z₀ 基準） denom = A + B/Z₀ + C·Z₀ + D
S₁₁ = (A + B/Z₀ − C·Z₀ − D) / denom
S₂₁ = 2 / denom
S₁₂ = 2·(AD − BC) / denom
S₂₂ = (−A + B/Z₀ − C·Z₀ + D) / denom

ABCD 行列は Z₀ に依存しません。上式で Z₀ を指定することで S パラメータに戻せます。Z₀ を変えれば同じ ABCD から異なる測定系の S パラメータを得られます。

🔬 直列接続の応用 — De-embedding（脱埋め込み）
  • Step 3 の復習：部品を直列接続すると行列の積になる — F_total = F₁ × F₂ × …
  • 実際の測定では DUT の前後に治具（ケーブル・コネクタ・基板など）が挟まる
  • VNA が計測するのは「治具 ＋ DUT」の合成特性であり、DUT 単体の値ではない：
    F_meas = F_jig × F_dut
  • 治具の F 行列が既知なら、逆行列をかけるだけで DUT を取り出せる：
    F_dut = [F_jig]⁻¹ × F_meas
💡 逆行列とは？
行列 F に対して F⁻¹ × F = 単位行列 I になるものを逆行列という。
2×2 行列では det = AD − BC を使って解析的に求められる： F⁻¹ = 1/det × [[D, −B], [−C, A]]

⚡ VIセンサー応用 — RF負荷のインピーダンス計測
  • VIセンサーとは、RFラインに挿入して電圧・電流を同時計測するプローブ。高電力RF装置全般において負荷インピーダンスをリアルタイムで監視するために広く活用されている
  • VIセンサー自体にもケーブル・CT・電圧プローブなどの寄生素子が存在し、測定値には治具の影響が含まれる
  • Step 4 の de-embedding を VIセンサーに適用することで、センサー自身の寄生を除去し、DUT の真のインピーダンスを得られる
  • RF電力計測が必要なあらゆる用途（産業用加熱・医療RF・通信試験など）に同じ手法が適用できる
